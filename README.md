# Cosmetic-webpage
A modern, responsive cosmetics web application built using React and Vite, focused on performance, scalability, and smooth user experience. The project demonstrates the use of component-based architecture, client-side routing, and animation libraries to create an interactive and visually appealing interface.

**#features**
Responsive and modern UI design 
Component-based architecture using React
Fast development and optimized builds with Vite
Client-side routing for seamless navigation
Smooth animations and transitions
Reusable and maintainable code structure
Cross-browser and multi-device compatibility

**Routing**
Implemented React Router DOM for client-side navigation
Enables smooth page transitions without full page reloads
Structured routes for Home, Products, and additional sections
Improves application performance and user experience

**Animations & Interactions**
Framer Motion is used for UI animations such as fade-ins, slide effects, and element transitions
GSAP (GreenSock Animation Platform) is used for scroll-based animations and advanced motion effects
Combined use of animations enhances visual appeal while maintaining performance
Animations are optimized to avoid layout shifts and improve engagement

**Tech Stack**
React – UI development using reusable components
Vite – Fast build tool and development server
JavaScript (ES6+) – Application logic
HTML5 – Semantic structure
CSS3 – Styling and responsive layout
React Router DOM – Navigation and routing
Framer Motion – UI animations
GSAP – Scroll-based and advanced animations
